## Payment Service
This is payment microservice of the Class Management System

## Requirements
* Open JDK 11
* Spring boot 2.7.9
* MySQL 8.0
* Maven 3.6



## Build

```
mvn clean compile package
```

## Run

```
mvn spring-boot:run
```

or

```
java -jar target/cms_payment_service-0.0.1-SNAPSHOT.jar
```

## Test

```
mvn test
```
